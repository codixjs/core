**像数 1, 2, 3 一样容易**

```bash
$ npm create codix-app ./example # 安装项目
$ cd example # 需要确定一个目录来存放资源
$ npm i # 安装依赖
$ npm run dev # 启动开发
$ npm run build # 启动编译
```
